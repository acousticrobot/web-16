These drawings are created with a heiroglyphic script based on toki pona.


Besides a gallery of images, this page connects to ordered lessons describing how this heiroglyphic writing can be understood and produced. There is also a pictographic dictionary which displays all of the glyphs, and contains links to instructional pages showing how to draw each glyph. Navigate through the lessons links on the left to learn more.
